## Description
Map `Opaque` runtime imports through generated Wasm adapters while keeping host objects outside linear memory and sharing one Host Object Table with the execution instance.

## Design
Treat `Opaque` as an abstract compiler/runtime ABI kind. Backend-private Wasm transport carries generational handles, runtime adapters resolve them through Runtime Context, and host bindings see only borrowed payloads or owned results. Do not serialize or expose the physical handle encoding.

## Acceptance Criteria
- [ ] Wasm compilation accepts Runtime Import signatures containing `Opaque`.
- [ ] Generated adapters transport handles without storing host references in linear memory.
- [ ] Wasmoon execution uses the execution instance's shared Host Object Table.
- [ ] Interpreter and Wasm agree on observable calls, limits, panic behavior, and normal finalization.
- [ ] Focused Wasm compiler and native Wasmoon tests pass.
