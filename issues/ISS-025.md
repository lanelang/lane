## Description
The semantic analysis index uses free helpers such as `span_contains` and `span_size` for repeated span behavior. Those operations should be attached to the span concept or a local wrapper so hover/definition code reads in domain terms.

## Design
Prefer methods on `SourceSpan` if MoonBit allows local extension for imported types in this package. If not, introduce the smallest local wrapper/helper receiver that avoids generic utility-style names. Preserve LSP hover/definition behavior.
