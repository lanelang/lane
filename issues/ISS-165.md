## Description
Implement `monadic-lift` after open-context resolution so bind continuations have a stable cold representation for resumed execution and an optimizable direct representation for normal returns.

## Design
Compute each bind continuation's free values and type parameters, lift its resumable form into a compiler-generated function, and retain an inline local form for the common direct-return path. Lower the bind protocol to select the lifted continuation only when execution actually resumes or yields. Expose captures to the existing closure lifting, ownership analysis, and ARC insertion stages.

## Acceptance Criteria
- [ ] Every resumable bind continuation has a lifted compiler-generated function.
- [ ] The normal-return path retains an inline continuation form.
- [ ] Lifted functions receive free values and type parameters explicitly.
- [ ] Multi-shot resume creates safe reusable ownership paths under ARC.
- [ ] Generic continuation captures preserve all required layout witnesses.
- [ ] Tests cover nested binds, captured values, generic captures, repeated resume, and ownership-sensitive continuation lifetimes.
