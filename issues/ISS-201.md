## Description
Replace the source `Write` runtime effect convention with an ordinary `Basic.Io.println` extern and update executable entry validation to the new `Io` contract.

## Design
Define `println : (String) -> Unit ! Io = extern("%println")`. Remove compiler and command recognition of `Basic.Io.Write.println`, Basic shape lookup, operation-call wrappers, and synthetic root handling from the active execution path. Accept only zero-argument entries returning `Unit` with effect `Empty` or exactly `Io`.

## Acceptance Criteria
- [ ] `Basic.Io.println` has the specified extern definition and public type.
- [ ] No compiler or command path recognizes `Basic.Io` or `%println` by source declaration shape.
- [ ] Pure and `Io` entries are accepted; residual algebraic effects and other entry shapes are rejected.
- [ ] A real source program using `Basic.Io.println` compiles, links, lowers, and runs.
- [ ] Focused command and Basic tests pass.
