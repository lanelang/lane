## Description
Carry extern bindings explicitly through Buslane lowering, module compilation, linking, executable elaboration, and LoisVM lowering so they become ordinary runtime-import function-table entries.

## Design
Model compiler intrinsics, extern bindings, and imported references as distinct origins at the owning boundaries. Externs lower through the existing synchronous primitive Runtime Import ABI v1. Builtins must lower intrinsically and never create a runtime import. Do not add continuations, a runtime-call opcode, or a second callable namespace.

## Acceptance Criteria
- [ ] Extern origin and symbol metadata survive compilation and linking without string-based reclassification.
- [ ] Reachable extern functions produce exactly the expected LoisVM runtime imports.
- [ ] Builtins produce no runtime imports.
- [ ] Interpreter and Wasm loaders use the existing shared import-resolution contract.
- [ ] Focused source-to-bytecode and public runtime-import tests pass.
