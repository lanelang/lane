## Description
Lower schema-v2 Instance Globals and initializer execution to the LoisVM Wasm tier without a Wasm start function.

## Design
Store heterogeneous global roots in per-instance linear memory, make `lane.entry` run initializer, selected entry, and reverse cleanup, and preserve existing module-owned memory, allocator, limits, imports, and tail calls.

## Acceptance Criteria
- [x] Wasm compilation supports global metadata and both global instructions.
- [x] The entry wrapper implements the same lifecycle and failure categories as the interpreter.
- [x] Instance roots are not encoded as heterogeneous Wasm globals or a start function.
- [x] Cross-backend tests cover values, cleanup, limits, and initialization failures.
