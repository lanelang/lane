# Buslane Documentation

This directory contains Buslane-owned implementation design material.

- `buslane-core.md`: implementation-oriented core-language model.
- `roadmap-checklist.md`: concrete implementation checklist for Buslane-owned
  roadmap work.
- `adr/`: decisions that directly constrain Buslane representation,
  verification, interpretation, and lowering boundaries.
- `contexts/`: glossary slices copied here because they name Buslane concepts
  used by this repository.

Normative language rules belong in the `spec` repository.
