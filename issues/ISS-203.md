## Description
Delete obsolete runtime-effect-operation metadata and synthetic runtime symbols from current module, linked, executable, and LoisVM lowering models.

## Design
Remove operation signature tables from compilation, fingerprinting, artifact codecs, linking, executable models, and lowering APIs. Bump every persisted schema whose layout or effect encoding changes and reject old versions through existing unsupported-version diagnostics rather than dual-decoding old semantics.

## Acceptance Criteria
- [x] Runtime-effect-operation tables and operation-derived runtime symbols are absent from current models and execution paths.
- [x] Fingerprints and codecs encode only the current extern-based runtime dependency model.
- [x] Affected artifact versions are bumped and obsolete versions are rejected explicitly.
- [x] Artifact roundtrips inspect complete values and preserve typed `Io` where applicable.
- [x] Focused compile, link, executable, lowering, and artifact tests pass.
