## Description
Make public effect set aliases usable across modules through existing transparent type alias interface artifacts.

## Design
Do not add a new artifact section. Public effect set aliases remain `type_aliases` with Effect-kind bodies. Public alias bodies may mention only interface-visible effects and aliases. Artifact text/binary roundtrip, inspect output, fingerprints, and imported-interface freshening must preserve enough information for downstream expansion.

## Acceptance Criteria
- [ ] `pub type UI = { Console, Log }` appears in interface inspection as a transparent alias.
- [ ] Downstream modules can use `Upstream.UI` in effect annotations.
- [ ] Public aliases that mention private effects are rejected.
- [ ] Interface artifact roundtrip preserves Effect-kind alias bodies.
- [ ] Fingerprints change when exported effect alias bodies change.
