## Description
Complete mon-trans by replacing private invokes, installs, latent effects, and selective binds with explicit handler dictionaries, ordered context arguments, answer-type CPS functions, and ordinary continuation callables.

## Design
Generate one typed dictionary shape per concrete effect, companion kind-Type/value parameters for abstract effect parameters, and answer-type CPS signatures for every monadic function. Lower invokes to statically selected operation callables and installs to lexical dictionary substitution. Clause bodies use the outer context; resume closures capture the inner context; final clauses use the outer context. Leave only abstract `OpenContext` plans for ISS-164.

## Acceptance Criteria
- [ ] Every non-pure function has the documented answer-type CPS signature and every pure function retains its direct signature.
- [ ] Invokes become statically selected dictionary callable invocations with no runtime operation tag.
- [ ] Installs construct immutable dictionaries and disappear.
- [ ] Clause, resume, and final contexts implement deep multi-shot semantics.
- [ ] Effect parameters have ordinary companion type/value parameters.
- [ ] Output contains no `Invoke`, `Install`, `Bind`, Buslane effect expression, or nonempty latent effect.
- [ ] Tests cover nested handlers, handler-arm effects, multi-shot resume, generic effects, and first-class effectful callables.
