## Description
Improve the semantic analysis path so hover and go-to-definition can still answer useful queries when the current document has unrelated parse, resolve, or type errors.

## Design
Start with the successful-check path for ISS-007 and ISS-008, then extend analysis to expose partial resolved or checked information when diagnostics are present. Prefer returning partial results over failing the entire LSP request. Keep diagnostics and analysis results in one response object so callers can make consistent decisions.

## Acceptance Criteria
- [ ] Analysis can return diagnostics plus partial semantic index data.
- [ ] LSP definition and hover still work for unaffected definitions in a file with unrelated errors.
- [ ] Tests cover at least one unresolved-value error where another reference still supports hover or definition.
- [ ] Unsupported or ambiguous positions still return `null`.
