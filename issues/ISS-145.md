## Description
`modules/lanec/compile/artifact_binary` is now an independent package, but its
files still carry the old parent-package prefix (`artifact_binary_*.mbt`). This
is no longer needed and makes the package read like it was mechanically moved
rather than designed as a cohesive package.

## Design
Rename files inside `modules/lanec/compile/artifact_binary` to package-local
names such as `core.mbt`, `common.mbt`, `buslane.mbt`, `elaborate.mbt`,
`interface.mbt`, `type.mbt`, and `error.mbt`. Do not change public APIs or
binary encoding. This is a naming/locality cleanup only.

## Acceptance Criteria
- [x] `artifact_binary` source filenames no longer repeat the package name as a prefix.
- [x] Public package API and generated interface remain behaviorally unchanged.
- [x] Validation passes with `moon check --warn-list +73`, `moon test --warn-list +73`, and `tools/check-lane-run-examples.sh`.
