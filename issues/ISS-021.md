## Description
Refactor compiler helper functions that already behave like methods into explicit MoonBit methods, and introduce traits only where multiple unrelated compiler node types genuinely share one capability used by generic code.

## Design
Prefer behavior-preserving changes. Do not add compatibility free-function aliases. Do not method-ize generated parser code, lexer byte predicates, one-off local helpers, or pure algorithms without a clear owning type. The first pass should prioritize diagnostics, artifact text writing, artifact validation/conversion, analysis span helpers, and canonical Buslane text rendering.
