## Description
Replace ambiguous `{}` expressions reported by MoonBit warning 0082 with explicit empty map or JSON object construction in diagnostic and LSP code and tests. The LoisVM completion work requires a warning-free cross-target build before further implementation.

## Design
Use `Map([])` where the expected value is a map and `Json::empty_object()` where the expected value is a JSON object. Preserve behavior and avoid unrelated LSP refactoring.
