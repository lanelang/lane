## Description
Execute schema-v2 Instance Globals and the Instance Initializer in the LoisVM interpreter with the agreed single-shot lifecycle.

## Design
Allocate uninitialized per-instance global roots, run initializer then entry under one configuration, enforce initialization phase rules, borrow globals without implicit retain, and release initialized owners in reverse order on cleanup-capable exits.

## Acceptance Criteria
- [x] Interpreter executes initializer before entry and supports both global instructions.
- [x] Duplicate initialization, pre-initialization reads, and incomplete initialization report internal runtime failure.
- [x] Normal and cleanup-capable failure paths release initialized globals exactly once.
- [x] Tests cover trivial, reference, callable, and erased global values.
