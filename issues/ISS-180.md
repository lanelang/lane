## Description
Represent `M_F<H>` and construct ordinary relay dictionaries that suspend residual operations into that Church-encoded computation.

## Design
Generate a residual computation type and pure injection for a handler result. For each concrete residual term, build a dictionary at that computation answer whose fields return a computation that invokes the caller-supplied dictionary at its actual answer and recursively runs the captured continuation.

## Acceptance Criteria
- [x] Empty residual rows simplify to direct handler results.
- [x] Concrete residual rows produce answer-polymorphic computations.
- [x] Relay fields preserve operation order, payloads, hidden witnesses, and reusable continuations.
- [x] Relay output is ordinary effect-free Buslane.
