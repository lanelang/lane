## Description
Add trusted type-and-effect synthesis for handler-elaborated effect core so mon-trans can classify expressions and calls without duplicating ad hoc type inspection. The input is already verified Buslane, but the private `Invoke` and `Install` forms need equivalent synthesized facts.

## Design
Return the exact value type and canonical evaluation effect for every private expression. Reuse Buslane metadata and substitution APIs, preserve left-to-right effect unions, remove handled effects from an install body, and include final and operation-clause effects. Treat missing metadata or impossible verified shapes as explicit compiler errors rather than fallback types.

## Acceptance Criteria
- [ ] Every handler-elaborated expression form synthesizes a value type and canonical effect.
- [ ] Calls include callee evaluation, argument evaluation, and latent call effects.
- [ ] Invokes include payload effects and their declared concrete operation effect.
- [ ] Installs remove handled effects from the body and include final/clause effects.
- [ ] Type application substitutes both ordinary and effect arguments.
- [ ] Exact tests cover pure calls, open effects, invokes, installs, and invalid trusted-core shapes.
