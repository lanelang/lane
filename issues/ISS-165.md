## Description
Implement `monadic-lift` after open-context resolution so every local selective-CPS bind continuation becomes an ordinary lifted function and closure accepted by the effect-free Buslane pipeline.

## Design
Validate the open-resolved compiler-private program and convert every generated local continuation into an ordinary Buslane `Function`/`TypeLambda`, producing an ordinary effect-free `@buslane.Program`. Lane does not generate Koka-style yielding and inline variants because it has no global yielding side channel. Preserve free value and ordinary type references in those local functions so the existing ANF-to-LoisVM closure lowering remains the single owner of capture analysis, layout witnesses, environment construction, ownership analysis, and ARC insertion.

## Acceptance Criteria
- [ ] Every resumable bind continuation becomes an ordinary compiler-generated Buslane function.
- [ ] Pure functions remain direct style and monadic functions use explicit answer-type CPS.
- [ ] Free values and ordinary type parameters remain visible to existing closure capture analysis.
- [ ] Multi-shot resume reaches the existing reusable closure and ARC paths.
- [ ] Generic continuation captures reach the existing layout-witness capture path.
- [ ] Tests cover nested binds, captured values, generic captures, repeated resume, and ownership-sensitive continuation lifetimes.
