```sh
moon -q run --target native modules/lane -- run examples/valid/13_existential_structs.lane:packed_int --lib-dir ../basic
```

Observed output:

```text

    actual type#11

Preserve the existing source-level existential rules; the fix should make Buslane metadata, hidden witnesses, and payload types agree.
