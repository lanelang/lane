## Description
Remove control-flow and value-transfer scaffolding that remains after ANF lowering and before ownership finalization.

## Design
Run VM CFG simplification before ARC insertion. Fold constant branches, thread jump-only blocks, merge compatible blocks, remove unused block parameters, propagate trivial values, eliminate unreachable blocks, and coalesce non-overlapping values into compatible slots. Preserve edge ownership metadata required by finalization.

## Acceptance Criteria
- [x] Unreachable and jump-only blocks are removed when safe.
- [x] Trivial block parameters and transfers are simplified.
- [x] Constant branches and repeated scalar constants fold locally.
- [x] Compatible non-overlapping values share slots without changing cleanup kinds.
- [x] VM CFG verification and ARC finalization pass after optimization.

## Close Notes
Added a pre-ARC VM CFG simplifier inside finalization. It propagates local trivial copies and repeated scalar constants, folds known Boolean/tag branches, removes dead non-trapping scalar work and unused block parameters, threads empty jumps, merges no-argument single-predecessor blocks, and compacts unreachable blocks to stable block IDs. A conservative interval allocator coalesces exact-compatible trivial values in single-block functions while retaining distinct owned/borrowed identities. The focused Bool-match regression falls from four bytecode blocks to three; the Coroutine Scheduler finishes with 1,195 bytecode slots and 152 post-ARC blocks from an initial 1,206 VM values and 148 blocks. Verification: VM CFG 14/14, LoisVM lowering 35/35, interpreter 21/21, Wasm 45/45, Lane 56/56, all runnable examples, warning-enabled check, Scheduler JIT/interpreter output `1112211322`, and `git diff --check`.
