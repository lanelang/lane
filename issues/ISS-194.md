## Description
Route source execution, linking, linked artifacts, and runtime loading through Whole-Program Elaboration and schema-v2 LoisVM images.

## Design
Keep `.lmo` as Buslane module objects and `.lbp` as final LoisVM bytecode. Move target convenience APIs to orchestration, update inspect output and artifact tests, and ensure `lane run` and `lane link` share the same executable path.

## Acceptance Criteria
- [ ] `lane run` and `lane link` invoke Whole-Program Elaboration exactly once.
- [ ] `.lbp` roundtrips schema-v2 bytecode and `lane inspect` reports initializer and globals.
- [ ] No command or module-link package bypasses Executable Program.
- [ ] Command, artifact, and runtime integration tests pass.
