## Description
Checked analysis currently traverses operation call arguments and handler arm
bodies, but it does not attach the source operation name to the resolved
operation symbol. This means goto-definition and hover work for the surrounding
arguments but not for the operation being performed or handled.

## Design
Use checked operation symbols to add reference and hover entries at operation
call sites and handler operation arms. Derive the visible operation-name span
from the source text and the checked expression or arm location.

## Acceptance Criteria
- [x] Operation calls expose a reference from the call-site operation name to the operation declaration.
- [x] Handler operation arms expose a reference from the arm operation name to the operation declaration.
- [x] Qualified operation owners continue to resolve as effect references through resolved analysis.
- [x] LSP tests cover goto-definition or hover at both operation call sites and handler arms.
