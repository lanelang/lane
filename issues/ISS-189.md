## Description
Add the target-independent `lanec/executable` package and replace the shallow linked executable wrapper with a Buslane-level Executable Program.

## Design
Whole-Program Elaboration accepts a Linked Program and selected entry, establishes ordered top-level initialization, runtime boundary policy, and the Execution Root Set, and returns the sole input consumed by execution-image targets. It does not perform target reachability collection.

## Acceptance Criteria
- [ ] `lanec/executable` owns Executable Program and Whole-Program Elaboration.
- [ ] Effect and runtime-boundary elaboration moves out of LoisVM-specific lowering.
- [ ] The package has no dependency on LoisVM bytecode, VM CFG, interpreter, or Wasm.
- [ ] Focused tests cover entry selection, initializer roots, and unused runtime operations.
