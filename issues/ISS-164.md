## Description
Implement `open-resolve` over the mon-trans output so abstract effect-context opening caused by effect subsumption becomes concrete handler-context adaptation before monadic lifting.

## Design
Compare the callee's expected effect context with the caller's current context. Remove identity openings and lower non-identity openings to explicit projection, reordering, masking, or empty-context operations over the Lane handler-context representation selected by ISS-162. Preserve the adapted context across both normal returns and resumed continuations. This pass does not create handlers or decide which operation clause runs.

## Acceptance Criteria
- [ ] Identical source and target effect contexts remove the abstract opening.
- [ ] Empty, singleton, and multiple-effect source contexts adapt explicitly and deterministically.
- [ ] Nested occurrences of the same effect preserve the intended handler level.
- [ ] Generic and polymorphic effect contexts preserve the information required for runtime adaptation.
- [ ] Context restoration remains correct across normal return and resumed continuation paths.
- [ ] No abstract effect-context opening remains in the pass output.
