## Description
Provide shared compiler-private CFG predecessor, reachability, dominator, use-definition, and liveness analyses over VM CFG function bodies.

## Design
Model VM CFG values as SSA-like definitions at inputs, block parameters, and instruction destinations. Treat edge arguments as predecessor terminator uses and target block parameters as definitions. Compute reachable blocks and immediate dominators from block zero, and compute backward block liveness with instruction-level live-after queries without changing ownership semantics. Replace duplicate local counting and reachability logic in existing simplification and devirtualization passes where behavior remains identical.

## Acceptance Criteria
- [x] CFG predecessors, reachability, dominators, and immediate dominators are computed.
- [x] Every value records its definition and deterministic use sites.
- [x] Block live-in/live-out and instruction live-after queries handle joins and loops.
- [x] Existing VM CFG passes reuse the shared analyses without behavior changes.
- [x] Focused diamond, loop, block-parameter, and unreachable-block tests pass.
- [x] No new public package interface is introduced.
