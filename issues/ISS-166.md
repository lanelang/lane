## Description
Connect the completed effect-erasure passes to the production Lane-to-LoisVM path. The current lowering entry calls a primitive runtime-effect rewrite before ANF and rejects residual source handlers and resumes.

## Design
Run handler elaboration, `mon-trans`, `open-resolve`, and `monadic-lift` before ordinary ANF and VM CFG lowering. Lower the remaining compiler-private bind, handler-context, and continuation protocol into ordinary functions, calls, data, and closures. Retain a narrow final bridge that converts validated residual outer runtime operations into ordinary runtime-import calls using the existing synchronous primitive v1 ABI. Remove or rename the temporary `erase_effects_before_anf` implementation so the code and documentation do not imply that primitive rewriting is complete effect erasure.
