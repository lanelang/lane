## Description
Replace runtime-import arity-only descriptors with complete direct-value parameter and result signatures supporting primitives plus `Opaque`.

## Design
Serialize ABI major, parameter kind array, result kind, and opaque lookup key. Expand Lane aliases and erase External Types to `Opaque` in LoisVM lowering. Runtime linking validates every kind but intentionally does not preserve External Type fingerprints or payload brands.

## Acceptance Criteria
- [ ] Runtime Import descriptors contain complete parameter and result kinds.
- [ ] Bytecode encode, decode, disassembly, validation, and tests use the current schema only.
- [ ] Lanec lowering derives direct ABI signatures from reachable extern source types.
- [ ] Interpreter and Wasm import resolution reject kind mismatches before execution.
- [ ] Arity-only descriptor and resolution paths are removed.
