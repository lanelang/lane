## Description
Prove that completed mon-trans produces only the residual forms owned by open-resolve and monadic-lift and preserves the required handler semantics.

## Design
Add a structural invariant validator and ordinary-execution semantic fixtures. Reject residual Invoke, Install, Bind, effect-kind type parameters, and nonempty latent effects while allowing only explicit ordinary contexts, local continuation functions, and abstract OpenContext plans.

## Acceptance Criteria
- [ ] The validator rejects every forbidden residual form with a typed compiler error.
- [ ] Successful output contains no Invoke, Install, Bind, source handler table, or nonempty latent effect.
- [ ] Only abstract OpenContext plans retain effect-row information.
- [ ] Structural tests cover generic effects and first-class effectful callables.
- [ ] Semantic tests cover nested handlers, handler-arm effects, final clauses, and multi-shot resume.
- [ ] All targets pass with zero warnings and the public interface diff is reviewed.
