## Description
Construct lexical handler dictionaries, lower body/final/clause code into the handler-local residual computation, and eliminate `Install` completely.

## Design
Use the relay contexts from ISS-180 for residual effects, instantiate handled dictionaries at `M_F<H>`, wrap source resume binders as transformed residual computations, lower the body with the nearest lexical dictionary, and finally run `M_F<H>` against the actual outer contexts and continuation.

## Acceptance Criteria
- [ ] `Install` becomes ordinary dictionary construction, functions, matches, and calls.
- [ ] Clauses and final code use outer contexts while resumes retain the inner dictionary.
- [ ] Nested same-effect handlers select the nearest dictionary.
- [ ] Multi-shot, handler-arm effects, generic clauses, and hidden parameters pass semantic tests.
