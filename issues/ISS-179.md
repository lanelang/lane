## Description
Replace every compiler-private `Invoke` with a statically selected operation callable from the current concrete effect dictionary.

## Design
Match the lexical dictionary's single constructor, bind its fields in declaration order, select the field owned by the operation metadata, apply hidden type witnesses, and call it with transformed payloads plus the current continuation.

## Acceptance Criteria
- [x] No lowered invocation carries an `OperationId` or runtime operation tag.
- [x] Operation order determines the selected immutable field.
- [x] Generic effect arguments and hidden operation witnesses instantiate the field correctly.
- [x] The resulting ordinary Buslane fragment passes verification.
