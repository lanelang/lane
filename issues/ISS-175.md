## Description
Rewrite non-pure functions and calls to the explicit answer-type CPS ABI while retaining direct style for pure functions.

## Design
Use the typed context schema to add ordered context parameters, transformed source parameters, and one continuation parameter to every monadic function. Instantiate answer types and pass explicit context values at monadic calls. Rewrite effect parameters to companion type/value pairs and preserve first-class effectful callables as ordinary answer-polymorphic values.

## Acceptance Criteria
- [ ] Pure function definitions and calls retain their direct ABI.
- [ ] Non-pure functions receive answer, ordered context, and continuation parameters with Empty latent effect.
- [ ] Monadic calls instantiate the answer type and pass contexts before source arguments and continuation.
- [ ] Effect-polymorphic functions receive ordinary companion type/value parameters.
- [ ] First-class and nested effectful callable types remain callable after rewriting.
- [ ] Strict left-to-right evaluation and shared match continuations remain explicit.
