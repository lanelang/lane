## Description
After diagnostic ownership is moved to phase-local typed implementations, add tests and lightweight source checks to keep the architecture from regressing into a new central wording switch.

## Design
Extend the diagnostic golden corpus where needed, keep LSP JSON assertions focused on message/data shape, and add source-level checks if practical to prevent `lanec/compile` from reintroducing parser/resolve/typecheck wording switches.
