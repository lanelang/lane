## Description
Make LoisVM lowering consume Executable Program, collect its transitive execution-image closure, and lower ordinary top-level values into an initializer and Instance Globals.

## Design
Treat every ordered top-level initializer and the selected entry as roots. Retain only transitively required functions, externals, and runtime imports. Reuse ordinary ANF, closure lifting, ownership analysis, ARC insertion, and slot allocation for initializer code.

## Acceptance Criteria
- [x] LoisVM lowering accepts one Executable Program rather than parallel linked inputs.
- [x] Non-function top-level terms produce ordered global initialization instead of `UnsupportedTopLevel`.
- [x] Unused functions and runtime imports do not enter the bytecode image.
- [x] ARC ownership for global initialization and borrowing passes VM CFG finalization tests.
