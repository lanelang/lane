## Description
Resolve the overlap between the planned monadic-lift capture transformation and the existing ANF-to-LoisVM closure lifting implementation.

## Design
Make monadic-lift the compiler-private effect IR to ordinary Buslane boundary. It validates and converts generated local continuation functions without introducing a second closure representation. Existing LoisVM lowering remains the single owner of free-value analysis, free runtime type parameters, layout witness capture, environment shapes, `MakeEnv`, `MakeClosure`, and ARC-visible ownership.

## Acceptance Criteria
- [x] Identify the existing closure-lifting owner and its capture/witness responsibilities.
- [x] Define monadic-lift output as ordinary effect-free Buslane.
- [x] Avoid a duplicate explicit closure or environment representation before ANF.
- [x] Update ADR-0067 and ISS-165 to reflect the actual pipeline boundary.
