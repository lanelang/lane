## Description
Preserve or improve inspection workflows for public values and artifacts after `lane run` and `lane runobj` stop debug-rendering arbitrary public entries.

## Design
Route public value and artifact inspection through `lane inspect` or an explicitly inspect-oriented command surface. Do not keep debug rendering as a fallback in `run/runobj`. If current inspect output cannot show the desired value or entry information, extend inspect rather than weakening executable entry validation.

## Acceptance Criteria
- [ ] Users have a clear inspect path for public entries or linked artifacts after run/debug rendering is removed.
- [ ] `run/runobj` diagnostics point users toward inspect tooling when they select a non-executable value.
- [ ] Tests cover the new diagnostic or inspect handoff behavior.
- [ ] Documentation no longer describes `run/runobj` as arbitrary public value rendering.
